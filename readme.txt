Créez un environnement Virtuel dans backend avec la commande python -m venv nom_de_l_environnement 

Activez la avec venv/scripts/activate (sur pc)

Installez le requirements avec la commande : pip install requirements.txt

Créez un superuser avec la commande! python manage.py createsuperuser 

Lancez le server avec python manage.py runserver

Ouvrez un autre terminal

Entrez dans le dossier frontend

lancez la commande npm install

Lancez le serveur en faisant: npm run dev

ET ENFIN PROFITEZ DE L'APP
